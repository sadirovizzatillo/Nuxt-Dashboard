# Nuxt Dashboard

A simple admin dashboard built with Nuxt 3, Ant Design Vue, and Tailwind CSS.

## Features

- Dashboard with stats, charts, and activity feed
- Users management (CRUD)
- Products management (CRUD)
- Multi-language support (English, Uzbek, Russian)
- Authentication with login page

## Tech Stack

- Nuxt 3
- Ant Design Vue
- Tailwind CSS
- Pinia (state management)
- Chart.js (charts)
- i18n (translations)

## Setup

```bash
# install dependencies
npm install

# run dev server
npm run dev

# build for production
npm run build
```

## API

Uses [DummyJSON](https://dummyjson.com) as a mock API for users and products data.
